4.3.1.1 将左递归转化为右递归

公式
对于形如
P→Pα∣β\mathrm{P} \rightarrow \mathrm{P \alpha} \mid \betaP→Pα∣β
的文法，可以通过公式化为非左递归的形式
P→βP′P′→αP′∣ε (  为空字) \begin{array}{l}P \rightarrow \beta P^{\prime} \\ P^{\prime} \rightarrow \alpha P^{\prime} \mid \varepsilon \quad \text { ( } \text { 为空字) }\end{array}P→βP′P′→αP′∣ε (  为空字) ​
一个栗子
更普遍的我们对于形如
P→Pα1∣Pα2∣⋯∣Pααn∣β1∣β2∣⋯∣βnP \rightarrow P \alpha_{1}\left|P \alpha_{2}\right| \cdots\left|P \alpha_{\alpha_{n}}\right| \beta_{1}\left|\beta_{2}\right| \cdots \mid \beta_{n}P→Pα1​∣Pα2​∣⋯∣Pααn​​∣β1​∣β2​∣⋯∣βn​
我们可以转化为
P→β1P′∣β2P′∣⋯∣βnP′P′→α1P′∣α2P′∣⋯∣αmP′∣ε\begin{array}{l}{P} \rightarrow \beta_{1} \mathrm{P}^{\prime}\left|\beta_{2} {P}^{\prime}\right| \cdots \mid \beta_{n} {P}^{\prime} \\ {P}^{\prime} \rightarrow \alpha_{1} {P}^{\prime}\left|\alpha_{2} P^{\prime}\right| \cdots\left|\alpha_{m} {P}^{\prime}\right| \varepsilon\end{array}P→β1​P′∣β2​P′∣⋯∣βn​P′P′→α1​P′∣α2​P′∣⋯∣αm​P′∣ε​
也就是所有β\betaβ后添P′P'P′，所有PαP\alphaPα交换加′'′，也就是αP′\alpha P'αP′

4.3.1.2 消除一个文法的一切左递归

其实也不是一切文法都可以
有一个小小的条件
如果文法不含回路（形如P⇒+P\mathbf{P} \stackrel{+}{\Rightarrow} \mathbf{P}P⇒+P)，也不含以ε\varepsilonε为右部的产生式。
WHY
利用以上公式还不可以消除一切的左递归，考虑这样一个文法
S→Qc∣cQ→Rb∣bR→Sa∣a\begin{array}{l}S \rightarrow Q c \mid c \\ Q \rightarrow R{b} \mid {b} \\ {R} \rightarrow { Sa\mid a }\end{array}S→Qc∣cQ→Rb∣bR→Sa∣a​
其并不直接满足上述的公式，但是如果尝试展开文法就可以得到
S⇒Qc⇒Rbc⇒SabcS \Rightarrow Q c \Rightarrow R b c \Rightarrow \mathrm{Sabc}S⇒Qc⇒Rbc⇒Sabc
我们将R代进Q的有关候选：
Q→Sab∣ab∣bQ \rightarrow S a b \mid a b \mid b Q→Sab∣ab∣b
再把RQ代进S的有关候选有：
S→Sabc∣abc∣bc∣cS \rightarrow S a b c\mid a b c\mid b c \mid cS→Sabc∣abc∣bc∣c
我们称这种代入法为RQS排序的代入法，后面将对更多代入法进行讨论。
进而就可以利用 4.3.1.1 中的方式
P→βP′P′→αP′∣ε (  为空字) \begin{array}{l}P \rightarrow \beta P^{\prime} \\ P^{\prime} \rightarrow \alpha P^{\prime} \mid \varepsilon \quad \text { ( } \text { 为空字) }\end{array}P→βP′P′→αP′∣ε (  为空字) ​
就可以将其转化为
S→abcS′∣bcS′∣S′S′→abcS′∣ε\begin{array}{l}S \rightarrow a b c S^{\prime}\left|b c S^{\prime}\right| S^{\prime} \\ S^{\prime} \rightarrow a b c S^{\prime} \mid \varepsilon\end{array}S→abcS′∣bcS′∣S′S′→abcS′∣ε​
此时一整个文法就变成了
S→abcS′∣h′S′cS′S′→abcS∣εQ→Sab⁡∣ab∣bR→Sa∣a \begin{array}{l}S \rightarrow a b c S^{\prime} \mid h^{\prime} \mathrm{S}^{\prime} \mathrm{cS}^{\prime} \\ S^{\prime} \rightarrow \mathrm{abcS} \mid \varepsilon \\{\color{Gray} Q \rightarrow \operatorname{Sab}|a b| b } \\ {\color{Gray} \mathrm{R} \rightarrow \mathrm{Sa} \mid a} \end{array}S→abcS′∣h′S′cS′S′→abcS∣εQ→Sab∣ab∣bR→Sa∣a​
我们可以发现，此时的QQQ，RRR是从SSS开始无法达到的，所以关于QQQ和RRR的规则是多余的，我们可以直接舍去。
注意，这里对于非终结符的排序会导致得到的算法不同，在上面的例子中，我们采用了RQS的顺序。
假如我们用SQR的顺序再来一次（也就是把S代进Q里，再把SQ代进R里），会得到以下文法。
