两步操作

  1. (cx)=(cx)-1
  2. (cs)=0?
    Y:向下执行
    N:转至标号处执行程序
标号代表一个地址

框架

  mov cx,循环次数
s:
  循环执行的程序段
  loop s

示例

计算2122^{12}212

assume cs:code
code segment
  mov ax,2
  
  mov cx,11
s:add ax,ax
  loop s
  
  mov ax,4c00h
  int 21h
 code ends
 end

123*236

简单来说，就是把123加236次
如果想要优化算法，可以把236加123次
DATAS SEGMENT
    ;此处输入数据段代码  
DATAS ENDS

STACKS SEGMENT
    ;此处输入堆栈段代码
STACKS ENDS

CODES SEGMENT
    ASSUME CS:CODES,DS:DATAS,SS:STACKS
START:
    MOV AX,DATAS
    MOV DS,AX
    
    MOV CX,236
    MOV AX,0
  s:ADD AX,123
    LOOP s
    
    MOV AH,4CH
    INT 21H
CODES ENDS
    END START
