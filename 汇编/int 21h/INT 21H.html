1H 读入 一个字符

接受参数：AH=1H

2H 输出 单个字符

接受参数：AH=2H

如果要输出的是数字，记得先变成ASCLL码
mov ah,2H
mov dx,SenLen
add dx,48
int 21H

9H 输出 以'$'为结尾的字符串

接受参数：AH=9H

mov ah,9H
mov dx,offset string1
int 21H

0AH：输入 读取缓冲字符数组

接受参数：AH=0AH

注意定义 keyword的第一个数字代表输入的最大长度 第二个数字会在输入后被存入输入的字符串长度 第三个字节之后是初始化的值
datas segment
  keyword db 16,0,16 dup('$')    
datas ends


codes segment
    assume cs:codes,ds:datas,ss:stacks
start:
    mov ax,datas
    mov ds,ax
    
    
      mov ah,0AH
      mov dx,offset keyword
      int 21H
      
    
    mov ah,4ch
    int 21h
codes ends
    end start
