JCXZ

格式全称中文测试条件
JCXZ OPRjump if CX register is zeroCX寄存器为0cx=0

检测点9.2

补全程序，利用jcxz指令，在内存2000H段中查找第一个为0的单元，将其偏移地址存储在dx中。
assume cs:code
code segment
start:mov ax,2000h
    mov ds,ax
    mov bx,0
s:
    mov cl,ds:[bx];
    mov ch,0;
    jcxz ok;
    inc bx;  
    jmp short s 
ok:
    mov dx,bx
    mov ax ,4c00h
    int 21h
code ends
end start
