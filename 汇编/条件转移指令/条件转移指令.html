格式全称中文测试条件
JZ(JE) OPR jump if zero(equal)结果为0ZF = 1
JNZ(JNE) OPR jump if not zero(not equal)结果非0ZF = 0
JS OPR jump if sign结果为负SF = 1
JNS OPR jump if not sign结果为正SF = 0
JO OPR jump if overflow溢出则转移OF = 1
JNO OPR jump if not overflow不溢出则转移OF = 0
JP OPR jump if parity even奇数PF = 1
JNP OPR jump if not parity (odd)偶数PF = 0
JC OPR jump if carry进位为1CF = 1
JNC OPR jump if not carry进位为0CF = 0
状态位
用来记录执行结果的状态
  • Z（zero）F（结果为0）
  • S（sign) F（正负符号）
  • O (overflow) F（结果溢出）
  • C (carry) F（无符号数最高位进位值）
