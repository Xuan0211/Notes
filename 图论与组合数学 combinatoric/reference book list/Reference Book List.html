Some books you could read, if you would like another viewpoint (although you are not required to read any of these books).
Most of these books will discuss some topics which are not in the course, so while you can read them from the start to the end, you will learn many extra things (that is not, of course, a bad thing!)
  • Shahriar Shahriari's "An invitation to Combinatorics" is a quite fun book to read, with many interesting examples. Some of the questions I use are based on questions in this book.
    An Invitation to Combinatorics (Shahriar Shahriari) (Z-Library).pdf
  • Some people really like "Discrete Mathematics with Ducks" by Sarah-Marie Belcastro. It has many jokes, and ducks, so read this only if that sounds fun to you! 有些人非常喜欢Sarah-Marie Belcastro的《带鸭子的离散数学》。它有很多笑话和鸭子，所以只有在听起来很有趣的情况下才读这个!?
    Discrete mathematics with ducks (Belcastro, Sarah-Marie) (Z-Library).pdf
  • T. Kyle Peterson's "Inquiry-Based Enumerative Combinatorics" is good, and many Universities use it as their text.
    Inquiry-Based Enumerative Combinatorics One, Two, Skip a Few... Ninety-Nine, One Hundred (T. Kyle Petersen) (Z-Library).pdf
  • Peter J. Cameron's "Combinatorics" is good, particularly if you would like a very mathematical analysis of the problems we discuss. Some parts may be hard to read if you are not happy with a lot of maths notation.
    Combinatorics Topics, Techniques, Algorithms (Peter J. Cameron) (Z-Library).pdf
