多路复用的概念
  • 复用的目的：将多路信号合并在同一信道上而互不影响，同时能在接收端彼此分离开来。
  • 频分多路复用FDMA
    • 介质带宽，比传送一路信号所需的带宽要宽得多
    • 频分多路复用技术：如何充分利用信道带宽，同时传输多路信号
    • 信道逻辑划分为n个较小带宽的子信道 频谱搬移：选择n路信号，利用频率不同的载波进行调制。 每路子信道传输一路调制后的信号。 防护频带--为了使n路信号各不相干扰覆盖，在每路信号的频段之间增加格外的频带。 接收端，利用不同频段的接收滤波器，将各路信号恢复。
  • 时分多路复用TDMA
    • 能完全用数字电路实现，广泛地使用。
    • 把时间帧划分成若干时隙，各路信号占用各自时隙
    • 一个信道可以有多少路信号复用主要取决于信道的带宽和每路信号的需求
    • 分类：同步时分多路复用（TDM）和异步时分多路复用（统计时分复用，STDM）
      • 同步时分多路复用（TDM）：指分配给每个终端的时间片是固定的，不管终端是否有数据发送，该终端的时间片都不能被其他终端占用。
      • 统计时分多路复用（STDM）：允许动态的按需分配时间片，需要发送数据的终端必须提出申请才能获得所需的时间片，否则时间片可以被其他终端占用。 优缺点：充分利用信道，控制较为复杂。 异步时分多路复用的输出数据速率比输入数据速率要小，当输入超过容量高峰时，需在复用器上加设缓冲器以容纳临时超出的输入数据。
  • CDMA(Code Division Multiple Access)码分多路复用
    • 允许所有站在同一时间使用整个信道进行数据传送，采用码型来区分各路信号。 特点：不再有冲突假设,信号可以线性叠加.
    • 在CDMA中，每个比特时间再分成m个码片，每个站分配一个唯一的M比特码序列。 当某个站发送“1” 时，在信道中发送它的码序列； 当发送“0”时，就发送它的码序列的补码(非值)。 这种编码方式使得发送信息量为原来的m倍.
试题2.15（2分）为什么拨号接入使用的调制解调器（56kpbs）速度慢？
每条回路到了端局后被滤波限制在3100HZ之间，将300HZ以下、3400HZ以上频率滤除。
P109
调制解调器之所以如此慢的原因是，电话的发明是为了承载人类的语音，而且整个电话系统也为了这个目的而被小心地优化过了。数据就好像不是自己亲生的子女，一直没有被认真地对待。每条本地回路都在端局那里被终止，就在这个终止的点上。有一个滤波器把所有300Hz以下、3400Hz以上的频率都减弱了。截止的地方并不尖锐---300Hz 和3400Hz都是3分贝点一一所以，尽管两个3分贝点之间的距离是3100Hz,但是，这一段带宽也常常被当作4000Hz来引用。因此，数据也被限制在这一窄窄的频段之中。
电话工业为什么使用56kbps？
电话信道的带宽大约是4000Hz(包括防护频段)。因此，每秒钟的最大独立采样次数为8000。在美国，每个采样的位数是8，其中1位被用于控制的用途，因而每秒钟允许传输用户数据56000位。 在欧洲，所有这8位数据都可以给用户使用，所以64kbPs的调制解调器也已经在使用了，但是，为了有一个国际性的标准约定，故选择了56000。
试题2.16（6分，频分、时分、码分，每对一个给2分）列举常见多路复用方法并对各个方法做简要比较。
  1. 频分多路复用（FDM）：
    1. 将频率分成若干个子频带，每个信道占用不同的频带进行传输。各个信道传输的数据不会相互干扰，接收端通过滤波器将各个信道的数据分离出来。
    2. FDM技术适用于模拟信号或数字信号，具有传输带宽大，支持多种不同速率信道的优点。但是，对于高速数据传输，需要精确的频率稳定性，频率稳定性差会影响信号传输质量。
  2. 时分多路复用（TDM）：
    1. 将时间分成若干个时隙，每个时隙只分配给一个信道使用。各个信道的数据按时序交错传输，接收端通过解复用器将数据分离出来。
    2. TDM技术适用于数字信号，具有简单、可靠、实时性好的优点。但是，TDM技术需要精确的时钟同步，时钟同步不好会影响信号传输。
  3. 码分多路复用（CDM）：
    1. 将不同用户的数据进行编码，不同用户之间的数据通过不同的编码进行区分。接收端通过解码器将数据分离出来。
    2. CDM技术适用于数字信号，具有抗干扰性好、安全性高的优点。但是，CDM技术需要昂贵的设备支持，对于大规模应用来说成本较高。
  4. FDM适用于模拟信号，TDM适用于数字信号，CDM适用于对数据安全性要求较高的应用场景。
试题2.17解释信道、信号、信道带宽、码元、波特率、比特率（6分，每解释对一个给1分）。
  • 信道（Channel）：是指信息传输的媒介，可以是物理媒介（如光纤、无线电波等）或者是逻辑媒介（如计算机内部数据通信）。
  • 信号（Signal）：是指携带信息的波形，可以是模拟信号或数字信号。
  • 信道带宽：最小衰减的情况下能够通过这种介质的频率范围；物理特性；单位：Hz
  • 波特率：每秒钟采样的次数；每秒传输的码元数量，单位是波特（Bd）。
  • 码元：每个采样发送一份信息；
  • 位传输率（比特率）：一条信道上发送的信息的数量=每秒采样数*每个采样的位数；
试题2.18对比阐述电路交换、报文交换与分组交换的特点。（6分，每解释对一个给2分）
  • 电路交换：
    • 两台计算机或数据终端在相互通信时使用一条实际物理链路，在通信过程中自始至终使用该链路进行信息传输，且不允许其他计算机或终端同时共享该链路的通信方式。
    • 电路交换通信过程：建立电路连接——传输数据——电路拆除
    • 优点：
      • 传输延迟小，唯一的延迟是物理信号的传播延迟；
      • 一旦线路建立，便不会发生冲突。
    • 缺点：
      • 建立物理线路所需的时间比较长。
      • 在电路交换系统中，物理线路的带宽是预先分配好的。对于已经预先分配好的线路，即使通信双方都没有数据要交换，线路带宽也不能为其他用户所使用，从而造成带宽的浪费(电话系统按时间收费)
  • 报文交换（包交换）：
    • 不事先建立物理电路，将发送数据作为整体交给中间交换设备，中间交换设备将报文进行存储，然后选择一条合适的空闲输出线将数据转发给下一个交换设备，如此循环往复直至将数据送到目的结点。
    • 分组交换：
      • 是报文交换的一种改进，将报文分成若干个分组，每个分组长度设置上限值。
      • 有限长度的分组使每个节点所需的存储空间减少，同时分组可以存储到内存中，提高交换速度。
      • 相同目的地址的分组选择不同路径结点
      • 到达目的节点的顺序可能不同，因此分组到达J后要重新按分组序号排序，以保证报文的有序接收。
    • 优点：
      • 电路利用率高。由于报文或分组可以分时共享两个节点之间的通道。
      • 在报文交换网中，通信量大时仍然可以接收报文或分组，但传输延迟会增加。
      • 可以把一个报文或分组发送到多个目的地。
      • 可以进行速度和代码的转换。
试题2.19对比阐述数据报与虚电路的特点。（这里选做，在学网络层知识时，必须弄懂）
数据报和虚电路的特点如下：
  • 数据报：是一种无连接的服务，每个分组独立选择路由，不需要建立和释放虚电路。数据报的优点是主机随时可发送数据，线路利用率高，容错能力强；缺点是分组可能丢失、重复或乱序，传输时延大且不确定，需要在目的地重新组装报文。 数据报适用于数据量较小或可靠性要求不高的场景，如网页浏览、文件传输等。
  • 虚电路：是一种面向连接的服务，在数据传输之前必须通过呼叫建立一条虚电路，在数据传输结束后必须释放虚电路。 虚电路的优点是分组按固定路由顺序传输，保证了分组的有序到达和可靠性；缺点是需要额外的开销来建立和维护虚电路，线路利用率低，故障恢复困难。 虚电路适用于数据量较大或实时性要求高的场景，如电话通话、视频会议等。

  • 奈奎斯特定理：C = 2H log2 V（位/秒）

试题2.28（3分）一个由振幅和相位构成的平面星座图有如下坐标点：（0,1）、（0,2）。请问一个具备这些参数的调制解调器是使用相位调制还是振幅调制？

试题2.30（4分）比较使用以下方案的无噪声4kHz 信道的最大数据传输率:
(a) 每次采样2 比特的模拟编码（比如QPSK）
(b) T1 PCM 系统
无噪声信道的最大数据传输率可以用奈奎斯特定理计算，即 R = 2B log2 V，其中 R 是数据传输率，B 是信道带宽，V 是每个符号的可能状态数。 对于 (a) 每次采样2 比特的模拟编码（比如QPSK），V = 4，B = 4kHz，所以 R = 2 × 4 × log2 4 = 16 kbps。 对于 (b) T1 PCM 系统，V = 128（因为每个样值用7 比特表示），B = 4kHz，所以 R = 2 × 4 × log2 128 = 56 kbps。
试题2.31（4分）三个分组交换网络每个包含n个节点。第一个是具有中心交换机的星型拓扑结构网络，第二个是双向环形网络，第三个是完全互联（或全连接，即任意两个节点之间都有线路相连）网络。请问，从传输路径的跳数来看，哪个最好？哪个最差？
星型：最好为2，最差为2，平均为2；
环形：最好为1，最差为n/2，平均为n/4；
如果考虑n为奇偶数，则n为奇数时，最坏为(n-1)/2，平均为(n+1)/4；n为偶数时，最坏为n/2，平均为n/4
全连接：最好为1，最差为1，平均为1

试题2.32（4分）请比较一下在一个电路交换网络中和一个（负载较轻的）分组交换网络中，沿着k跳的路径发送一个x位消息的延迟情况。电路建立的时间为s秒，每一跳的传播延时为d秒，分组的大小为p位，数据传输率为b 比特每秒（bps）。在什么条件下分组网络的延时比较短？
对于电路交换，t = s时电路建立起来；t = s + x/b_时报文的最后一位发送完毕；t = s + x/b + kd_时报文到达目的地。
对于分组交换，t = x/b时报文的最后一位发送完毕；为到达最终目的地，最后一个分组必须被中间的路由器转发k-1次，每次重发花时间p/b，所以总的延迟为t = x/b + (k-1)p/b + kd。
为了使分组交换比电路交换快，必须：
x/b + (k-1)p/b + kd < s + x/b + kd
所以：
s > (k-1)p/b
试题2.33 （4分）假定x位用户数据将以一系列分组形式，在一个分组交换网络中沿着一条共有k跳的路径向前传输，每个分组包含p位数据和h位的头，这里x远大于p+h。线路的传输率b 比特每秒（bps），传播延时忽略不计。请问，什么样的p值将使总延时最小？
所需要的分组总数是x/p，因此总的数据加上头信息交通量为(p+h)x/p位。
源端发送这些位需要时间为(p+h)x/pb
中间的路由器重传最后一个分组所花的总时间为(k-1)(p+h)/b
因此我们得到的总的延时为
f(p) = (p+h)x/pb + (k-1)(p+h)/b
对该函数求p的导数，得到：
hx/p2b+(k−1)/bhx/p^2b + (k-1)/bhx/p2b+(k−1)/b
令
hx/p2b+(k−1)/b=0hx/p^2b + (k-1)/b = 0hx/p2b+(k−1)/b=0
得到
hx/p2=k−1hx/p^2 = k-1hx/p2=k−1
因为p>0，所以
p = √hx/k-1
此时能使总的延迟最小。
