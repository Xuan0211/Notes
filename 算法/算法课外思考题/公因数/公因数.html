方法一：辗转相除法

辗转相除法，又称 欧几里得算法，对于两个非负整数 x,y (x>y)x,y(x>y)x, y \ (x > y)x,y (x>y)x,y (x>y)x,y(x>y)，其计算公式为：
gcd⁡(x,y)=gcd⁡(y,x mod y) \gcd(x, y) = \gcd(y, x\ mod\ y) \ gcd(x,y)=gcd(y,x mod y)

代码

int main()
{
  //while实现辗转相除
  int a ,b;
  int r;
  cin>>a>>b;
  if(b>a) swap(a,b);
  r=a%b;
  while(r)
  {
    a=b;b=r;
    r=a%b;  
  }
  cout<<b;
  return 0;   
} 
int gcd(int a,int b)
{
  return b==0?a:gcd(b,a%b);
}

方法二：更相减损术

更相减损术，出自《九章算术》（原本是为了求约分，修改后可用来求解最大公约数）的一种求最大公约数的算法。

思路
  • 以较大的数减较小的数，接着把所得的差与较小的数比较，并以大数减小数。
  • 继续上述操作，直到所得的减数和差相等为止。

代码与实现

      while（a != b）
      { 
        while(a>b)
        {
          a = a -b;
        }
        while(b>a)
        {
         b = b-a;
        }
      }
      return a * pow(2，num);

方法三：Stein算法

Stein算法 是一种计算两个数最大公约数的算法，是针对欧几里得算法在对大整数进行运算时，需要试商导致增加运算时间的缺陷而提出的改进算法。

思路

对于给定的两个正整数x,yx, yx,y
  1. 判断它们是否都是偶数：
    若是，则用2约简，直至两者不同时为偶数。
  2. 将两数中的偶数（如存在）进一步约简为奇数；（更相减损）
    以较大的数减较小的数，接着把所得的差与较小的数比较，并以大数减小数。
  3. 继续上述第2步操作，直到所得的减数和差相等为止。
  4. 最终，第1步中约掉的若干个2的积与第3步中等数的乘积就是所求的最大公约数。

代码

      //一点点优化
      while (a % 2 == 0 && b % 2 == 0)
      {
        a = a >> 1;
        b = b >> 1;
        num++;
      }
      while（a != b）
      { 
        while(a>b)
        {
          a = a -b;
        }
        while(b>a)
        {
         b = b-a;
        }
      }
      return a * pow(2，num);
