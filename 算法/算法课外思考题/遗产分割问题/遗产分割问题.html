John有两个孩子，在John病逝后，留下了一组价值不一定相同的魔卡，现在要求你设计一种策略，帮John的经管人将John的这些遗产分给他的两个孩子，使得他们获得的遗产差异最小（魔卡不能分拆）。
【动态规划之背包问题】——子集背包（416. 分割等和子集）_yours_棒棒糖的博客-CSDN博客

问题转换BY YSQ：

转换成0-1背包问题，每张魔卡的体积是V，价值也是V，背包的容积为sum/2，怎么使得背包中物品的价值最大

代码

for (int i = 0; i < n; i++)
    {
        for (int j = sum / 2; j >= 0; j--)
        {
            if (j > v[i])//防止下表越界
                dp[j] = max(dp[j], dp[j - v[i]] + v[i]);
        }
    }
