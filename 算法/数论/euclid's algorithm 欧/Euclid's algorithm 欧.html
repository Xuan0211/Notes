欧几里得算法

辗转相除法，又称 欧几里得算法，对于两个非负整数 x,y (x>y)x,y(x>y)x, y \ (x > y)x,y (x>y)x,y (x>y)x,y(x>y)，其计算公式为：
gcd⁡(x,y)=gcd⁡(y,x mod y) \gcd(x, y) = \gcd(y, x\ mod\ y) \ gcd(x,y)=gcd(y,x mod y)

代码

int main()
{
  //while实现辗转相除
  int a ,b;
  int r;
  cin>>a>>b;
  if(b>a) swap(a,b);
  r=a%b;
  while(r)
  {
    a=b;b=r;
    r=a%b;  
  }
  cout<<b;
  return 0;   
} 
int gcd(int a,int b)
{
  return b==0?a:gcd(b,a%b);
}
