数位DP

数位dp是一种计数用的dp，一般就是要统计一个区间  [A , B ] 内满足一些条件数的个数。
所谓数位dp，字面意思就是在数位上进行dp。
数位的含义：一个数有个位、十位、百位、千位......
数位dp的实质就是  换一种暴力枚举的方式，使得新的枚举方式满足dp的性质，然后记忆化就可。
算法提高课（试听课）—— 数位DP_哔哩哔哩_bilibili

例题

多组数据，每次给定区间 [n,m] ，求在n到m中没有 “62“ （连续）或“4“ 的数的个数。
如62315包含62，88914包含4，这两个数都是不合法的。
0< n<=m < 1000000

状态定义

设一个数组 dp[ i ][ j ]，表示 i 位数，最高位是j 的符合题意的个数。

状态转移

  • 如果j 是4，肯定都不符合条件（因为题目不让有4），所以直接是0；
  • 如果 j不是6，那么它后面随便取，只要符合题意就可以，所以 dp[ i- 1 ][ k ] ,k可以随便取的和；
  • 如果 j是6，后面只要不是2 就行，所以是 dp[ i-1 ][ k ],   k除了2都可以，求和即可
谁是高位谁是低位取决于遍历的方向不同
int maxlen=7;//最大开到七位数
for(int i=1;i<=maxlen;i++)
{
  for(int j=0;j<=9;j++)
  {
    if(j==4}
      dp[i][j]=0;
    else
    {
      if(j==2)
      {
        for(int k=0;k<=9;k++)
        {
          if(k!=2)
            dp[i][j]+=dp[i-1][k];
        }
        else
        {
          for(int k=0;k<=9;k++)
          {
            dp[i][j]+=dp[i-1][k];
          }
        }
      }
    }
  }
}
统计不含7的
int main()
{
//数位DP+倍数直接运算
  int dp[10][10];
  memset(dp, 0, sizeof(dp));
  for (int i = 0; i <= 9; i++)
  {
    if (i != 7)
      dp[0][i] = 1;

  }
  int n;
  scanf("%d", &n);
  int c;
  c = 0;
  int nn = n;
  while (nn)
  {
    c++;
    nn /= 10;
  }
  for (int i = 1; i <= c; i++)
  {
    for (int j = 0; j <= 9; j++)
    {
      if (j == 7)
      {
        dp[i][j] = 0;
      }
      else
      {
        for (int k = 0; k <= 9; k++)
        {
          dp[i][j] += dp[i - 1][k];
        }
      }
    }
  }
  int ans = 0;
  nn = n;
  for (int i = c - 1; i >= 0; i--)
  {
    if (nn == 0)
    {
      break;
    }
    int tt;
    tt = (int)pow(10, i);
    for (int j = 1; j <= nn / tt; j++)
    {
      ans += dp[i][j];
    }
    nn %= tt;
  }
  return 0;
}
